# Seed — vault/collaborate/

> Collaboration artifacts: clarifying questions, external input, and Q&A logs.
> Read `CLAUDE.md` for what changed.

## What This Directory Serves

Houses documents used for collaboration — gathering input, answering clarifying questions, and coordinating with external stakeholders or contributors.

## Contents

| Dir/File | Description |
|----------|-------------|
| `clarifying-questions/` | Open questions sent to stakeholders for input |

## Architecture

This directory is a lightweight inbox/outbox for collaboration artefacts. Each clarifying question document captures:
- The question (context + specific ask)
- Who needs to answer
- The answer when received
- Decision made based on answer
