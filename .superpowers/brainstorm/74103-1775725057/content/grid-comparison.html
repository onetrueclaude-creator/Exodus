Neural Lattice Grid: Current vs Proposed

How should faction territory and node placement work?

CURRENT: Spiral Arms
Problems: nodes placed between grid squares, spiral arms twist and overlap, hard to read faction boundaries
PROPOSED: Quadrant Grid
Nodes centered on grid squares. 90-degree quadrants. Clean boundaries. Easy to read.

Which approach?

A

Quadrant Grid (Recommended)

4 quadrants at 90 degrees. Nodes snap to grid cell centers. No spiral, no arm connections. Clean, readable, game-like.

B

Quadrant Grid + Value Gradient

Same as A, but cells closer to origin glow brighter (higher value / density). Creates a natural "fight for the center" dynamic.

C

Quadrant Grid + Contested Borders

Same as A, but cells adjacent to quadrant borders are "contested" — any faction can claim them. Creates border conflict zones.
