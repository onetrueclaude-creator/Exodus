How should new nodes appear as blocks are mined?

Currently: 4 nodes spawn per block (one per faction arm). With quadrants, how should the grid grow?

Option A: Ring Expansion
Each mined block adds one Chebyshev ring of cells per quadrant. Grows outward symmetrically. Inner cells are more valuable (higher density).
Option B: All Cells Pre-exist
Fixed-size grid (e.g. 8x8 per quadrant). All cells visible from start. Players claim empty cells. Simpler — no "growth" mechanic, just territory grab.
