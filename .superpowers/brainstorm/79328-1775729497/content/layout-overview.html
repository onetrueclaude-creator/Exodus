Proposed Layout: Left Controls, Full Grid, Right Info

Clean three-zone layout with no overlap

Three-Zone Layout (Recommended)

Left: Dock rail + sliding panel (terminal, chat, stats). Center: Full-width grid with ResourceBar on top. Right: Nothing permanent — only floating tooltip when hovering/clicking a node. No overlap possible.

Interaction Rules

CLICKING A CELL
- Shows a small info tooltip near the cell
- Shows: faction, coords, density, owner
- NO actions (no mining, no claiming)
- Click elsewhere or Esc to dismiss
ALL ACTIONS VIA TERMINAL
- Mine: through agent terminal menu
- Claim: through agent terminal menu
- Deploy: through agent terminal menu
- Secure: through agent terminal menu
REMOVE
- "My Agents" dropdown (top-left)
- QuickActionMenu (left sidebar actions)
- AgentPanel (left sidebar detail)
- Right-side node popup
KEEP
- ResourceBar (top strip)
- TabNavigation (Network / Account)
- Dock rail (move to left side)
- Floating panels (terminal, chat, stats)
