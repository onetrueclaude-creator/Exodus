# Seed — vault/research/

> Research base: competitor analysis, market research, and user research.

## What This Directory Serves

Stores research artifacts that inform product and engineering decisions for ZK Agentic Network. Covers competitive landscape, market sizing, and user behavior research.

## Contents

| Dir/File | Description |
|----------|-------------|
| `competitors/` | Competitor landscape — blockchain games, social dApps, AI agents |
| `market/` | Market sizing, TAM/SAM/SOM, token economy research |
| `users/` | User research: behavior patterns, persona definitions, onboarding data |
| `_index.md` | Directory index |

## Key Research Areas

- Blockchain gaming (Axie Infinity, Star Atlas, Parallel) — how they handle resource economies
- AI agent platforms (character.ai, Replika) — agent interaction patterns
- Social dApps (Lens Protocol, Farcaster) — content primitives and social graphs
- Crypto game economics — token distribution, staking mechanics, play-to-earn sustainability
