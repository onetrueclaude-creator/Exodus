# Seed — vault/ideas/

> Ideas inbox: raw concepts, experiments, and future direction thinking.
> Read `CLAUDE.md` for what's been added and what's been promoted to features.

## What This Directory Serves

Captures raw, unfiltered ideas for ZK Agentic Network — game mechanics, monetization, social features, technology experiments. Ideas here are not commitments; they're possibilities under consideration.

## Contents

Currently empty — ideas are captured in `_templates/idea.md` format when added.

## Idea Lifecycle

```
vault/ideas/       → raw concept
    ↓ validated
vault/product/features/  → promoted to feature spec
    ↓ decided
vault/product/decisions/ → logged as decision
    ↓ built
src/ / apps/       → implemented
```
